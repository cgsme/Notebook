# 锁

## 共享锁、排他锁

共享锁（s）和排他锁（x）都是**行级锁**。

**共享锁**：也称为**读锁**，允许多个连接可以同时并发的读取同一资源，互补干扰。
**排他锁**：也称为**写锁**，一个写锁会阻塞其他的写锁或者读锁，保证同一时刻只有一个连接可以写入数据，同时防止其他用户对这个数据的读写。

**注意：共享锁、排他锁其实是锁机制本身的策略，通过这两种策略对锁做了区分。**

## Intention Locks （意向锁）
